06:45 抵达悉尼 SYD T1 后，跟着 Arrivals / Passport Control / Immigration 指示走，先完成入境澳洲。
入境后到 Baggage Claim 取行李
过 Customs / Biosecurity 后，会进入 T1 Arrivals Hall 到达大厅，通常从 Passenger Exit A 或 Passenger Exit B 出来。
出来后先找这几个英文标志：
Baggage Storage
Luggage Storage
Smarte Carte
Airport Information
Arrivals Hall
寄存时准备好护照，工作人员通常会要求登记姓名、联系方式、件数和取回时间。行李可能会做安全检查或爆炸物痕迹检测。付款一般用信用卡/手机支付。
寄存后一定要保留收据或取件凭证，并拍照留底。护照、钱包、手机、充电宝、相机、药品、签证材料、下一程登机资料不要寄存，随身带走。

出关进入 Arrivals Hall 后，跟着 Rideshare / Uber / Taxi 的标志走到 T1 外面的 Uber pickup zone。
打开 Uber 下单后，App 会给一个 6 位 PIN 码。
到 Uber 排队区后，不一定是找某一辆指定车，而是上前方第一辆可用 Uber，把 6 位 PIN 给司机。
上车前核对司机和车辆信息，再出发。

备注：这一天必须早起，不建议自驾去 Milford Sound。To-do：
出发前一天确认团是否正常运行。
准备雨衣、防风外套、晕车药、饮用水、简单零食。
手机和相机提前充满电。

# 10.03 Day 10（第 10 天）Te Anau、Queenstown
10:00-10:30 起床，早餐，退房。
11:00 Te Anau（蒂阿瑙 / 峡湾门户小镇）出发，返回 Queenstown（皇后镇 / 南岛旅游中心）。
13:30 抵达 Queenstown，午餐。
15:00 下午作为 Queenstown 补充时间，可安排前面没做成的温泉、缆车、湖边散步、购物或休整。
18:30 Queenstown 晚餐。
住宿建议：Queenstown
驾驶强度：轻松，约 2 小时
备注：这是砍掉西海岸后换来的缓冲日，Milford 之后不立刻长途赶路。To-do：
预订 10.03 Queenstown 住宿 1 晚。
如果 09.30 没订 Onsen / Skyline，可把这天作为补充备选。

# 10.04 Day 11（第 11 天）Queenstown、Cromwell / Omarama、Twizel / Tekapo
09:30-10:00 起床，早餐，退房。
10:30 Queenstown（皇后镇 / 南岛旅游中心）出发，往中部湖区方向回程。
12:00 Cromwell（克伦威尔 / 中奥塔哥小镇：以水果、葡萄酒产区和中途补给闻名，适合咖啡、加油、短暂停留）短暂停留，咖啡、补给或水果店。
14:00 Omarama（奥玛拉玛 / 公路中转小镇：位于 Queenstown 回 Mackenzie 湖区的路线上，适合午餐、加油、休息）/ Twizel（特威泽尔 / 中转住宿小镇）午餐。
15:30 可选 Lake Pukaki（普卡基湖 / 冰川湖：天气好时湖水蓝、雪山视野好）/ Tekapo（蒂卡波 / 湖边小镇）轻量补拍，按天气决定。
17:00-18:00 抵达 Twizel / Tekapo 入住。
住宿建议：Twizel 优先，Tekapo 备选
驾驶强度：中等
备注：这一天是回程移动日，尽量不要晚于 10:30 出发。To-do：
预订 10.04 Twizel / Tekapo 住宿 1 晚。
确认当天是否要住 Tekapo 看湖景，还是住 Twizel 更省心。

# 10.05 Day 12（第 12 天）Twizel / Tekapo、基督城
09:30-10:00 起床，早餐，退房。
10:30 Twizel（特威泽尔 / 回程中转小镇）/ Tekapo（蒂卡波 / 湖边小镇）出发，返回基督城。
13:00 Geraldine（杰拉尔丁 / 回基督城路上的休息小镇）/ Ashburton（阿什伯顿 / 坎特伯雷平原上的较大城镇：适合午餐、加油、超市补给，离基督城更近）午餐、休息。
16:00-17:00 抵达基督城机场附近，入住。
17:30 加油、整理车辆和行李，确认第二天凌晨还车流程。
住宿建议：基督城机场附近
驾驶强度：中等
备注：最后一晚必须住机场附近，不安排市区深度活动。To-do：
预订 10.05 基督城机场附近酒店。
确认酒店到租车还车点 / 机场航站楼的距离。
提前加满油。
确认租车 key drop 位置和凌晨还车流程。
整理行李，避免第二天凌晨临时翻箱。

# 10.06 Day 13（第 13 天）基督城、墨尔本、香港
key drop（自助还钥匙箱 / 非营业时间还车方式）流程：
凌晨把车开到租车公司指定还车区。
停好车。
拍照记录车身、油表、里程、停车位置。
拿走所有行李。
锁车。
把车钥匙投入租车公司的 key drop box（自助钥匙箱）。
之后租车公司上班后再验车、结算。
03:30 起床，退房，前往基督城机场。
04:00 左右还车。停好车后拍照记录车身、油表、里程、停车位置，再投入 key drop。
04:20 抵达 CHC（Christchurch Airport / 基督城国际机场），办理值机和出境手续。
06:20 基督城 CHC >> 08:10 墨尔本 MEL（Melbourne Airport / 墨尔本机场：澳洲主要国际机场之一，本次为中转机场）
澳洲航空 QF168（Qantas 航班号 / QF 是澳洲航空代码），飞行约 3h50m。08:10 抵达墨尔本，中转约 3h05m。
11:15 墨尔本 MEL >> 17:55 香港 HKG（Hong Kong International Airport / 香港国际机场）
澳洲航空 QF029（Qantas 航班号），飞行约 9h40m。17:55 抵达香港，后续返回广州。
住宿建议：无
驾驶强度：低，但需要凌晨早起
备注：前一晚一定要把油、行李、还车流程全部处理好。To-do：
确认 CHC >> MEL >> HKG 行李是否直挂香港。
确认墨尔本中转是否需要澳洲签证 / 过境条件。
保存租车还车照片，直到押金解除。
包车回广州。
